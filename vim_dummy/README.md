# LOOP FILES

The files in the vim_dummy directory are here just to serve as dummy data on which to do an
edit-loop while adjusting a vim config file (vimrc*, init.lua, init.vim, etc.).

This way, the vim config file can be open in one tmux pane/window and the edit-loop can be going on
in another pane/window, and the dummy file(s) will be opened and closed quickly and repeatedly to
test whatever tweaks have been made in the vim config.

See bin/edit-loop.
